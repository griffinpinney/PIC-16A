## Example 1

Consider the following code.

```python
L1 = [0]
L2 = L1

L1.append(1)
L2.append(2)

print(L1, L2)
```

After the first two lines.

```box-drawing
                     ┌   ┐
 L1 ───────────────► │   │
                     │ 0 │
 L2 ───────────────► │   │
                     └   ┘
```

After the next two lines.

```box-drawing
                     ┌         ┐
 L1 ───────────────► │         │
                     │ 0, 1, 2 │
 L2 ───────────────► │         │
                     └         ┘
```

## Example 5

Consider the following code.

```python
def trial(L):
    L.append(0)
    L = [1]

starts_empty = []

trial(starts_empty)
print(starts_empty)
```

After the argument-to-parameter assignment.

```box-drawing
                                     ┌  ┐
 starts_empty ─────────────────────► │  │
                                     │  │
                               ╭───► │  │
                               │     └  ┘
                               │
        ┌┄┄┄┄┄┄┄┐              │
        ┊       ┊              │
        ┊   L ─────────────────╯
        ┊       ┊
        ┊ scope ┊
        └┄┄┄┄┄┄┄┘
```

After `L.append(0)`.

```box-drawing
                                     ┌   ┐
 starts_empty ─────────────────────► │   │
                                     │ 0 │
                               ╭───► │   │
                               │     └   ┘
                               │
        ┌┄┄┄┄┄┄┄┐              │
        ┊       ┊              │
        ┊   L ─────────────────╯
        ┊       ┊
        ┊ scope ┊
        └┄┄┄┄┄┄┄┘
```

After `L = [1]`.

```box-drawing
                                     ┌   ┐
 starts_empty ─────────────────────► │ 0 │
                                     └   ┘

        ┌┄┄┄┄┄┄┄┐
        ┊       ┊                    ┌   ┐
        ┊   L ─────────────────────► │ 1 │
        ┊       ┊                    └   ┘
        ┊ scope ┊
        └┄┄┄┄┄┄┄┘
```

After the function scope has been destroyed and garbage collection has taken place.

```box-drawing
                                     ┌   ┐
 starts_empty ─────────────────────► │ 0 │
                                     └   ┘
```
